# DivePC
A Dive Computer Simulator

# Build and Run

## Build Windows/Linux/Mac OS X
Open command prompt in root folder where gradlew.bat is and type
"gradlew desktop:dist"
## Run Windows/Linux/Mac OS X
Open command prompt in root folder where gradlew.bat is and type
"gradlew run"
